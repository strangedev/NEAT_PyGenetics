\subsection{Detailed Description}
A thread which operates on networking message queues via \hyperlink{namespaceNEAT__PyGenetics_1_1NEAT_1_1Networking_1_1Server_1_1JSONSocket}{J\+S\+O\+N\+Socket} in two different modes\+: 


\begin{DoxyEnumerate}
\item receive The worker thread will accept \hyperlink{namespaceNEAT__PyGenetics_1_1NEAT_1_1Networking_1_1Server_1_1JSONSocket}{J\+S\+O\+N\+Socket} connections, receive dictionaries, creates command objects and inserts them into the appropriate queue (in\+\_\+queue)
\item send The worker thread will wait for command objects to appear inside the outgoing queue (out\+\_\+queue), convert them to dictionaries and send them to the client via a \hyperlink{namespaceNEAT__PyGenetics_1_1NEAT_1_1Networking_1_1Server_1_1JSONSocket}{J\+S\+O\+N\+Socket}. 
\end{DoxyEnumerate}
