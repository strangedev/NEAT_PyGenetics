28  A set of Nodes as the output layer. These are identical in number

31  A hidden layer. This consists of a list of nodes which are calcula-
32  ted in the order of the list. This list is taken from the topologi-

38  These are nodes that have outgoing links that close a cycle. These
39  nodes have to be treated separately, because the cycle closing edges

41  calculation value is used as the outgoing base value. If the genome
42  was not simulated yet. a default value is used as the initial value.

44  Usage:
