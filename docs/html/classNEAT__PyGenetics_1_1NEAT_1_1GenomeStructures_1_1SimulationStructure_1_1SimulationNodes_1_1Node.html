Detailed Description

A single node in a SimulationGenome.

Stores information about its successors and a current value and has methods for propagating its value to its successors.

Successors are stored in a set to prevent adding the same successor mul- tiple times.

Weights are stored separately and are accessed with the successor as the key. :param initial_value: Optional initial value. May or may not be over- written when starting a simulation-step. :return:
