Detailed Description

A class providing a high-level interface to a client connected via the NEATClient.

It receives commands from NEATServer's message queue und responds to them. When a function from SimulationConnector is called, the client is expected to have sent a certain command (if there are no commands in the message queue, the Server will wait for the next command to be received). If the client has sent another command, or a command where the expected parameters are not present, NetworkProtocolException will be raised.
