PyCreatures - Help
-------------------------------------------------------------------------------

h                           - print this help message

q                           - quit

map                         - display the map

clear [WIDTH  HEIGHT]       - generate a new world of a certain width and 
                              height

spawn AMOUNT ENTITY_TYPE    - spawn a certain amount if entities at random

anim CYCLES [DELTA_T]       - Advances the simulation by a certain amount of
                            - cycles. Prints the map for each cycle and waits
                              a certain amount of time between each cycle.

<ENTER>                     - Advance the simulation by 1 cycle

<NUMBER>                    - Advances the simulation by a certain amount of
                              cycles.

ENTITY_TYPES:               - Corn, Mouse, Cat, JohnCena
